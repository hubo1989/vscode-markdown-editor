## 🌟 v1.0.0 重大更新

### 📊 新增功能特性

#### 图表支持
- **🎨 Mermaid 图表** - 流程图、时序图、饼图、甘特图等
- **🏗️ PlantUML 图表** - UML类图、用例图、活动图、序列图等
- **📈 ECharts 可视化** - 柱状图、折线图、饼图、散点图、雷达图等
- **🎼 音乐记谱** - ABC记谱法五线谱和音乐符号

#### 数学公式支持
- **🧮 KaTeX 引擎** - 高性能数学公式渲染
- **数学公式块** - 支持 LaTeX 风格的数学表达式
- **行内数学公式** - 文本中的数学表达式
- **数学宏定义** - 可定制的数学快捷方式

#### 工具栏增强
- **主按钮设计** - 所有图表和数学工具作为主工具栏按钮
- **智能快捷键** - Cmd/Ctrl + Shift 快捷键支持
- **主题自适应图标** - 图标自动适配 VS Code 主题色
- **一键模板** - 预配置模板，提高效率

### 🔄 技术升级
- **vditor 3.11.2** - 最新稳定版本
- **TypeScript 5.9.3** - 最新类型安全特性
- **KaTeX 数学引擎** - 专业的数学公式渲染
- **多服务器支持** - PlantUML 备用渲染服务器
- **本地渲染模式** - 增强离线能力
